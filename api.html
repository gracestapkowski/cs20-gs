The API I selected is called the Dog API and it returns a random link to a dog image every time it is run.


I found this API on this github page.


I decided to ask for a image of a random dog regardless of breed, but this API can also return images of dogs based on a selected breed, like this: https://dog.ceo/api/breed/{yourbreed}/images.


One application where this API would be helpful would be on a website all about dogs with separate pages for each breed. You could display a random image of each breed to give a visual.

Another application would be on a "useless" webisite that people use for fun/comedic relief. There is a whole database of them here! Sometimes it can be fun to go through these websites and have a laugh, and some of the dog pictures that come up from this API are quite funny.
